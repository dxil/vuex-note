# vue-note

> 利用vuex构建一个记事本应用，地址[demo](http://dingxl.com/vuex-note/dist/)

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
